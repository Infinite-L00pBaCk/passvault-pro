Full Feature Documentation

Deep dive into the architecture and capabilities of PassVault Pro.

Robust Authentication

  • Local Accounts: Create personalized accounts with a unique username completely offline.
  • Dual Login Methods: Support for traditional master passwords or a quick-access 4-digit PIN.
  • Secure Hashing: All credentials are processed using SHA-256 hashing via the Web Crypto API before local storage.
  • Visual Personalization: Each user profile is assigned a unique color chip based on their account for easy switching.

Advanced Generation

  • Random Mode: Fully customizable with toggles for case, numbers, symbols, and an "Exclude Ambiguous" filter.
  • Passphrase Mode: Generates human-readable but cryptographically secure phrases based on dictionary wordlists.
  • Manual Mode: Input and audit the strength of pre-existing passwords before saving them to the vault.
  • Strength Indicator: Real-time visual feedback from "WEAK" to "STRONG" dynamically calculated.

Security Health Audit

  • Real-Time Analysis: Dedicated dashboard monitoring total, strong, weak, and reused passwords.
  • Risk Detection: Automatically flags weak passwords, reused credentials, and "aging" passwords.
  • Security Score: Provides a numerical score (0-100) and a qualitative grade based on overall vault health.

Vault Management

  • Categorization: Organize entries using icons for Web, Banking, Gaming, Social, Shopping, Work, and more.
  • Smart Search & Sort: Instantly filter entries by platform or username, and sort by date, name, or strength.
  • Metadata Tracking: Automatically logs creation dates and allows for custom notes or 2FA backup codes.
  • JSON/CSV Export: Securely backup your entire encrypted vault to a portable file offline.
